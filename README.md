# Edu Spark ⚡ - Student Management System

> A modern, dark-themed admin dashboard for efficient student and institution management.

Edu Spark is a comprehensive Student Management System designed to streamline administrative tasks for educational institutions. It features a sleek, intuitive dark UI that provides quick access to key metrics, student records, attendance tracking, and user management.

## ✨ Key Features

* **Secure Authentication:** User login and registration system.
* **Interactive Dashboard:** Real-time overview of total students, employees, revenue, and profit, along with an interactive calendar.
* **Attendance Tracking:** Visual representations (charts) of daily student attendance (Present, Late/Half leave, Absent).
* **Student Directory:** Comprehensive CRUD (Create, Read, Update, Delete) functionality to manage student records easily.
* **Profile Management:** Customizable user profiles with avatar uploads and account settings.
* **Responsive Dark UI:** A highly polished, eye-catching dark theme optimized for administrative workflows.
